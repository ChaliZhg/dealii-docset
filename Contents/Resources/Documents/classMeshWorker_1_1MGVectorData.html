Detailed Description

template<typename VectorType, int dim, int spacedim = dim>
class MeshWorker::MGVectorData< VectorType, dim, spacedim >

Based on VectorSelector, this is the class that implements the function VectorDataBase::fill() for a certain type of multilevel vectors, using AnyData to identify vectors by name.

Author
Guido Kanschat, 2010
