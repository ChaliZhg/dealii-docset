Purely virtual function. This function is used to advance from time t to t+ delta_t. F is a vector of functions that should be integrated, the input parameters are the time t and the vector y and the output is value of f at this point. J_inverse is a vector functions that compute the inverse of the Jacobians associated to the implicit problems. The input parameters are the time, , and a vector. The output is the value of function at this point. This function returns the time at the end of the time step.
