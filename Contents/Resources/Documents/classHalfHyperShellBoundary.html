Detailed Description

template<int dim>
class HalfHyperShellBoundary< dim >

Variant of HyperShellBoundary which denotes a half hyper shell where the first coordinate is restricted to the range (or ). In two dimensions, this equals the right half arc, in three space dimensions it is a half shell. This class might be useful for computations with rotational symmetry, where one dimension is the radius from the axis of rotation.

Author
Wolfgang Bangerth, 2000, 2009
