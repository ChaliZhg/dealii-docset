Detailed Description

A namespace in which we declare "extractors", i.e. classes that when used as subscripts in operator[] expressions on FEValues, FEFaceValues, and FESubfaceValues objects extract certain components of a vector-valued element. The result of applying an extractor to these objects is an object with corresponding type from the namespace FEValuesViews. There are extractors for single scalar components, vector components consisting of dim elements, and second order symmetric tensors consisting of (dim*dim + dim)/2 components

See the description of the Handling vector valued problems module for examples how to use the features of this namespace.
