Detailed Description

A class that implements the interface to use the PETSc Eisenstat preconditioner, which implements SSOR on the diagonal block owned by each processor.

See the comment in the base class PreconditionerBase for when this preconditioner may or may not work.

Author
Wolfgang Bangerth, Timo Heister, 2004, 2011
