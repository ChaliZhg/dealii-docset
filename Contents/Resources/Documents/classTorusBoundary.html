Detailed Description

template<int dim, int spacedim>
class TorusBoundary< dim, spacedim >

Class describing the boundary of the torus. The axis of the torus is the -axis while the plane of the torus is the - plane. A torus of this kind can be generated by GridGenerator::torus.

This class is only implemented for dim=2,spacedim=3, that is, just the surface.
