Detailed Description

template<typename PayloadBlockType>
class TrilinosWrappers::internal::BlockLinearOperator::TrilinosBlockPayload< PayloadBlockType >

This is an extension class to BlockLinearOperators for Trilinos block sparse matrices.

Note
This class does very little at the moment other than to check that the correct Payload type for each subblock has been chosen correctly. Further extensions to the class may be necessary in the future in order to add further functionality to BlockLinearOperators while retaining compatability with the Trilinos sparse matrix and preconditioner classes.
Author
Jean-Paul Pelteret, 2016
