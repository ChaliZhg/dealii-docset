Detailed Description

template<int dim>
class QAnisotropic< dim >

Quadrature formula implementing anisotropic distributions of quadrature points on the reference cell. To this end, the tensor product of dim one-dimensional quadrature formulas is generated.

Note
Each constructor can only be used in the dimension matching the number of arguments.
Author
Guido Kanschat, 2005
